# Arthnex — Engineer the Future

Official website for **Arthnex**, a software studio building digital products, SaaS platforms, and tools that people actually love using.

---

## Overview

Arthnex is a software studio founded by two engineers who believe the best products come from deep care about users, craft, and the long game. This site showcases our products, philosophy, team, and ways to connect.

Built with React + Vite, designed from scratch, and engineered with the same standards we apply to every product we ship.

---

## Tech Stack

- **React 18** — UI framework
- **Vite** — Build tool
- **CSS3** — Custom properties, animations, responsive layout
- **Google Fonts** — Syne (display), DM Sans (body), DM Mono (code)

## Getting Started

```bash
# Clone the repo
git clone https://github.com/arthnex/arthnex-website.git
cd arthnex-website

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
